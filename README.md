# Naga Pavithra Lagisetty's Portfolio

A modern, interactive portfolio website built with Three.js, showcasing my skills, experience, and projects.

## Features

- Interactive 3D background with particles and animated torus
- Responsive design that works on all devices
- Smooth scroll animations
- Modern, clean UI with glassmorphism effects
- Sections for education, skills, experience, and projects

## Setup

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm start
```
4. Open your browser and navigate to `http://localhost:5173`

## Technologies Used

- Three.js for 3D graphics
- GSAP for animations
- Vite for development and building
- HTML5 & CSS3 for structure and styling

## Customization

You can customize the portfolio by:

1. Modifying the content in `index.html`
2. Adjusting styles in `style.css`
3. Changing 3D elements in `main.js`

## Deployment

To build for production:
```bash
npm run build
```

The built files will be in the `dist` directory, ready for deployment to any static hosting service.
